各国株価指数正規化

データソース:YahooFinance API 更新予定:毎月1日

株価指数の考察

このグラフは、2000年10月16日を基準（100）として、各国の株価指数を正規化し、比較したものです。主要な市場の動向を示しており、それぞれの経済や株式市場の成長速度の違いが可視化されています。

主な観察点

  1. ジャカルタ総合指数（インドネシア）の著しい成長

    インドネシアのジャカルタ総合指数は、他の国と比較して圧倒的な成長を見せています。基準値（100）から約2000以上に達しており、インドネシアの経済成長や新興市場としての強いパフォーマンスが反映されています。インフラ投資、人口増加、国内消費の拡大が成長を支えた要因と考えられます。

  2. アメリカ（ナスダック・ダウ平均）の安定的な上昇

    ナスダックはITバブル崩壊後の回復から現在に至るまで、ハイテク産業の発展が大きく寄与しました。特に2010年代以降、テクノロジー企業（GAFAなど）の成長が顕著です。

  3. 日本（日経平均）の相対的な停滞

    日経平均は、長期的には緩やかな上昇を見せているものの、他の市場に比べて成長率が低いです。バブル崩壊後の低成長、人口減少、デフレなどが長期的な課題として影響しています。

  4. 韓国（KOSPI・KOSDAQ）の堅調な成長

    韓国の株式市場はハイテク・半導体産業の成長によって支えられています。ただし、2010年代後半には成長の鈍化が見られます。

  5. シンガポール（ストレーツタイムズ指数）の緩やかな上昇

    シンガポールの市場は安定していますが、他の新興市場（インドネシア）ほどの成長は見られません。成熟市場としての特徴が表れています。

  6. マレーシア（クアラルンプール総合指数）の横ばい傾向

    他の市場と比較して成長が限定的であり、原材料価格の影響や政治的な不安定さが影響していると考えられます。

全体的なトレンド

新興市場（インドネシア、韓国）は高い成長率を示しており、グローバル経済の牽引役になっています。一方で、先進国市場（日本、アメリカ）は成熟市場として安定的な動きを見せています。

考察と今後の注目点

  • インドネシアの持続可能性: ジャカルタ総合指数は著しい成長を遂げていますが、外資依存や経済構造の脆弱性が課題として残ります。
  • 日本の再活性化の可能性: 成長率が低迷している日本市場は、経済構造改革やデジタル化の推進が必要です。
  • グローバル経済の分極化: 新興市場と先進国市場の間で成長格差が広がっており、世界経済全体のバランスへの影響が注目されます。
